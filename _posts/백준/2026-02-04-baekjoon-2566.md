# 2566번 - 최댓값
```python
# 1차원 배열로도 해결 가능하지 않나?
a = []
for i in range(9):
    tmp = list(map(int, input().split()))
    for j in range(9):
       a.append(tmp[j])

max_index = a.index(max(a)) # 최댓값이 위치한 곳
max_row = (max_index // 9) + 1 # 일반적으로 세는 숫자에서의 행
max_col = (max_index % 9) + 1 # 일반적으로 세는 숫자에서의 열

print(max(a))
print(max_row, max_col)
```

행과 열의 수가 주어졌으니 1차원 배열로도 해결이 가능했습니다.
주어지지 않았더라도 1차원 배열 내에서 해결 가능할 것 같고요.
