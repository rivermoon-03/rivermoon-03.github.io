```python

# A = 아스키 65, Z = 90
# 즉 ord('A') = 65
# -55를 해 주어야 함.

# ZZZZZ 36이면?
# 36^0 * Z + 36^1 * Z + ...
# 답은? 



a, b = input().split()
# a = 수, b = 진법

sum = 0
p = len(a) - 1
for i in range(len(a)):
    if a[i].isdigit():
        sum += (ord(a[i]) - 48) * (int(b) ** p)
    else:
        sum += (ord(a[i]) - 55) * (int(b) ** p)
    p -= 1
print(sum)
```

진법 변환을 사람이 하는 것처럼 실시하였다.
p를 0으로 두고 문자열의 마지막부터 돌아가면서 p를 1씩 더하는 방법으로도 사용 가능하다.

파이썬에서는

```python
print(int(a, int(b)))
```

이 코드로 위의 로직을 모두 대체할 수 있다.

## 진법 변환 2 
```python
#10진법, 진법을 입력받으면
#그 진법에 맞게 수를 변환해야 한다
# 이전 문제에서 A부터 Z까지는 ord(a) - 55를 이용해서 10~35로 변환했었다.
# 그럼 이번에는 55를 더해주면 될 것이다. (숫자는 해당 없다.)

a, b = map(int, input().split())
# a는 10진법의 수이고, b는 진법이다.
# 60466175 36 -> ZZZZZ
c = [] # 거꾸로 저장해야 하기 때문에 리스트에 담아둔다.

while a > 0:
    if a % b < 10:
        c.append(a % b)
    else:
        c.append(chr(a % b + 55))
    a //= b

c.reverse()
print("".join(map(str, c)))



```

대학교 1학년의 전산학기초에서 배우는 진법 변환을 생각하면 쉽게 풀 수 있는 문제였다.
